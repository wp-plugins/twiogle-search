== Description ==

This adds a widget that lets you search from Twitter and Google at the same time.  Displaying Twitter Search results on the right
and Google Search Results on the left.  

You can also put in your Adsense Pub ID and Channel number and Twiogle will display advertisments under your account so you get the money for people that search
from your page.

== Installation ==

1.  Simply Copy the plugin to your plugin folder and active it.  
2.  Then Add the widget in the Appearance section.
